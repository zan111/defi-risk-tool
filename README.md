# DeFi Risk Tool

A Python dashboard for analyzing DeFi positions on Aave V3 (Ethereum chain).  

Features:
- Calculate Health Factor (HF) and Loan-to-Value (LTV)
- Run stress tests on your portfolio
- Visualize results with charts

Technologies:
- Python, Dash, Plotly, pandas
- Data fetched from Aave V3 subgraph (Ethereum)
